How to use?

It's pretty simple. Just add an image tag to your html and make let the source be https://api.corbie.dev/lorem-image/{width}/{height}/{color} where width and height are the dimensions you want and color is the hex code without the #. For example:

<img src="https://api.corbie.dev/lorem-image/600/400/ffd801">

Width and Height are currently limited to 3840px which pretty much covers every 4k display.

The image is of type PNG and is considered to stay the default type. WebP is currently evaluated and other formats might come aswell.
